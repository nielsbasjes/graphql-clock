Trivial GraphQL "clock" service

Purpose

The purpose of this service is to have a trivial service that can be used to experiment with doing various GraphQL operations.

GraphiQL

When the application is running, you can use GraphiQL as a web based GraphiQL client: GraphiQL client

Query

Example:

query {
    time {
        year
        month
        day
        hour
        minute
        second
        millisecond
        iso
    }
}

Subscriptions

Example:

subscription {
    time {
        year
        month
        day
        hour
        minute
        second
        millisecond
        iso
    }
}
